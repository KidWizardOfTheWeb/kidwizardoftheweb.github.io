## Limiter Cut Radio: Radio for All

#### March 2026-Present

Github repository: https://github.com/KidWizardOfTheWeb/LimiterCut.Radio

Work in progress.
Features to be added:
- Multicasting with host-casters and co-casters in tandem.
- Frontend implementation.
- Statistics DB for servers.

LC-Radio allows anyone to spin up their own audio channel for others to listen to as they stream audio. 
Users will also be able to host their own servers with minimal setup.
Local testing works, server-instance testing coming soon.

**Tech stack**:
- Full-stack
- Python
- Sockets
- APIs
- JWT tokens
- AWS EC2 instances
